World Wide Web

Die wichtigsten Technologien die das Web zum Erfog gemacht haben

  • Uniform Resource Locator (URL) und Uniform Resource Identifier (URI)
  • HyperText Markup Language (HTML)
  • HyperText Transfer Protocol (HTTP)

URL

Die allgemeinen Bestandteile einer URL sind:

  • Protokoll (z.B. http)
  • Hostname (z.B. www.example.com)
  • Dateiname (z.B. index.html)

Web Server

Ein Webserver ist ein Computersystem, das Anforderungen über HTTP verarbeitet (das grundlegende Netzwerkprotokoll, das verwendet wird, um Informationen über das World Wide Web zu verteilen). Die primäre Funktion eines Webservers ist die Speicherung, Verarbeitung und Bereitstellung von Webseiten an Clients. Die Kommunikation zwischen Client und Server erfolgt über das Hypertext Transfer Protocol (HTTP).

Inhalte werden auf ihren Datenbanken gespeichert. Auf dem Web-Server ist die HTML-Datei, die der Client bekommen kann, damit er die Seite sehen kann. Der Inhalt von Webseiten wird in HTML geschrieben.

Web Client

Webbrowser oder allgemein auch Browser sind spezielle Computerprogramme zur Darstellung von Webseiten im World Wide Web oder allgemein von Dokumenten und Daten. Neben HTML-Seiten können Webbrowser verschiedene andere Arten von Dokumenten wie zum Beispiel Bilder und PDF-Dokumente anzeigen. Webbrowser stellen die Benutzeroberfläche für Webanwendungen dar.

HTML

Der "html tag"

  • Er teilt dem Browser mit, dass es sich um ein HTML-Dokument handelt.
  • Er repräsentiert den Stamm eines HTML-Dokuments.
  • Er ist der Behälter für alle anderen HTML-Elemente (mit Ausnahme des !DOCTYPE tags).

DOCTYPE

Die !DOCTYPE Dekleration ist kein HTML-Tag. Sie ist eine Anweisung für den Webbrowser, über welche HTML Version die Seite geschrieben wird.

Hervorhebungen eines Textes

  • Der em tag: macht einen Text hervorgehobener. Wird meist kursiv dargestellt
  • Der strong tag: markiert wichtigen Text. Wird fett gedruckt dargestellt
  • Der b tag: macht den Text fett gedruckt

Die beiden wichtigsten HTML Elemente

Das Head Element kann einen Titel für das Dokument, Skripte, Stile, Metainformationen und vieles mehr enthalten. Das Body Element enthält alle Inhalte eines HTML-Dokuments wie Text, Hyperlinks, Bilder, Tabellen, Listen usw.
